Origen de los Alimentos

-- Introducción
La motivación de este laboratorio se basa en la utilización de datos abiertos publicados y a partir de ellos realizar una representación.

-- Los datos
En el caso de este proyecto se utilizaron datos sobre la Habilitación y Registro de Alimentos de Uruguay.
Los campos que fueron utilizados para el proyecto son:
	- Capitulo
	- Clase
	- Pais
	- Cantidad Productos
Estos datos fueron transferios a un archivo .csv con una estructura especifica para facilitar el funcionamiento.

Para la localización de los paises se utilizó datos sobre las coordenadas geográficas de cada país.

-- Software
El software utilizado apra la interfaz es Processing y la librería ControP5.
Para la representación se utilizó un modelo 3D del planeta y curvas de bezier que unen los distintos paises con Uruguay.
Luego para aplicar filtros de los Capitulos se utilizó ControlP5
