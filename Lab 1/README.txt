El proyecto trata de aprovechar el movimiento generado por usuario al jugar con el Kinect, adquiriendo sus gestos y almacenándolos para luego recuperarlos y generar con estos un discurso artístico, representado en una imagen pictórica.

INSTRUCCIONES DE USO:
Dentro de la solución MovementTracking de Visual Studio 2012 se encuentra el proyecto encargado de traducir los movimientos a xml.
Este proyecto debe ser recompilado y agregado como librería
La salida es un xml llamado "movements" el cual es utilizado por algunos de los programas de processing para dibujar la salida gráfica. Para esto es necesario ubicar el xml dentro de la carpeta data.
